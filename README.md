## Validación automática con GitHub Actions

Este proyecto implementa un workflow de GitHub Actions para realizar validaciones automáticas cada vez que se ejecuta un `push` o se crea un `pull request`.

El workflow verifica la existencia de los siguientes archivos esenciales del proyecto:

* index.html
* style.css
* script.js
* README.md

Si todos los archivos están presentes, la validación finaliza exitosamente. En caso contrario, el proceso falla y se notifica el error.

La implementación de GitHub Actions contribuye a mantener la calidad y consistencia del proyecto mediante la integración continua.
